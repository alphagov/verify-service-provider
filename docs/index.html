API Documentation

POST /generate-request

Parameters

Parameter In Type Description
body body object No description

Body parameter

{
  "levelOfAssurance": {
    "description": "Level of assurance requested by the Service",
    "type": "string",
    "enum": [
      "LEVEL_1",
      "LEVEL_2",
      "LEVEL_3"
    ]
  }
}

Responses

Status Meaning Description
200 OK An object containing a SAML request.
400 Bad Request An error due to a problem with creating the SAML request. Can be because of given parameters are not recognized.
500 Internal Server Error An error due to an internal server error.

POST /translate-response

Parameters

Parameter In Type Description
body body object An object containing a details of a SAML Authn response. (required)

Body parameter

{
  "samlResponse": {
    "description": "A SAML Authn response as a base64 string.",
    "type": "string",
    "format": "byte"
  },
  "secureToken": {
    "description": "A token that was generated for the original SAML Authn request. The token is used to verify that the request and response are from the same browser.",
    "type": "string",
    "format": "byte"
  }
}

Responses

Status Meaning Description
200 OK Response contains an object with details of a translated SAML response.
400 Bad Request an error due to a problem with translating the Response
401 Unauthorized Response identifies failures when authenticating the user
500 Internal Server Error an error due to an internal server error
